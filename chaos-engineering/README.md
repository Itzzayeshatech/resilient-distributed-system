\# Chaos Engineering Project



\## Overview

This project demonstrates chaos engineering using Kubernetes and Chaos Mesh.



\## Features

\- CPU stress testing

\- Memory stress testing

\- Auto-recovery system (self-healing)



\## Tech Stack

\- Kubernetes (Minikube)

\- Chaos Mesh

\- Node.js



\## How It Works

1\. Deploy application

2\. Inject chaos (CPU/Memory)

3\. Monitor failures

4\. Automatically recover using Node.js script



\## Commands



\### Apply Chaos

kubectl apply -f cpu-chaos.yaml

kubectl apply -f memory-chaos.yaml



\### Run Recovery

node monitor/app.js# Chaos Engineering Project
